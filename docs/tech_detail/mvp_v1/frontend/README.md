# MVP V1 前端实现总结

本目录用于记录 `docs/tech_plan/mvp_v1/frontend/` 各模块的实际落地情况。

当前已完成：

- `01_base_data.md`：基础数据模块（人员管理、龙虾管理）实现总结
- `02_template.md`：模板模块（模板列表、详情、流程发起）实现总结
- `03_flow_run.md`：流程实例模块（流程列表、详情、取消）实现总结
- `04_node_processing.md`：节点处理模块（节点工作台、动作按钮、评论附件日志）实现总结
- `05_collaboration_trace.md`：协同留痕模块（流程/节点评论、附件、节点日志）实现总结
- `06_deliverable.md`：交付模块（生成、列表、详情、验收）实现总结
- `07_pages.md`：页面与布局模块（工作台、统一导航、菜单分层）实现总结
- `08_fixed_flow_nodes.md`：固定流程节点模块（固定表单映射、节点校验、节点类型兜底）实现总结
- `09_scope_acceptance.md`：范围与验收模块（必做能力对齐、只读约束收口、非范围入口核对）实现总结
